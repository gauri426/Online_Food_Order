# Online Food Order

A responsive online food ordering website built using **HTML** and **CSS**.

## Features
- Responsive layout for desktop and mobile
- Hero section with featured items
- Food grid with images
- Navbar and footer
- Fully static front-end project (no backend)

## Tech Stack
- HTML
- CSS
- Images for products and layout
